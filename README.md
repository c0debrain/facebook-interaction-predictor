# facebook-interaction-predictor

Predicts likes, comment or total interactions of a facebook page post using machine learning

![alt-image](https://media.giphy.com/media/kOvaFAKLCIUTK/giphy.gif)

Using Adaboost regressor with decision tree to predict the interactions of a post such as Likes, comments and others. 

Using [ Facebook Metrics dataset](https://archive.ics.uci.edu/ml/datasets/Facebook+metrics) from UCI. The data is related to posts' published during the year of 2014 on the Facebook's page of a renowned cosmetics brand. [Download it](https://archive.ics.uci.edu/ml/machine-learning-databases/00368/)
